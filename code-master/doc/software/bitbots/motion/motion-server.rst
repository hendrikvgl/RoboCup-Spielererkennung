Motion Server
=============

.. important::
    Der Motion Server kümmert sich um die gesamte Motion und die verschiedenen Zustände in denen wir uns befinden
    können (z.B. Penalized). Er muss aber nicht umbedingt komplett verstanden werden um Code schreiben zu können.
    Außerdem werden viele Dinge (z.B. Animationen abspielen) aus höheren Verhaltensteilen eh nicht direkt auf dem
    Motionserver aufgerufen.
